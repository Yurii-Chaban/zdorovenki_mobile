Условия доставки

Уважаемые покупатели, заказывая товар в магазине "Здоровенькі були!", Вы имеете возможность совершить покупку наиболее удобным для Вас способом:

  • Посетив наш салон магазин медицинских товаров и реабилитационного оборудования по адресу г. Киев, ст. метро Черниговская, ул. Попудренко, 52, оф. 214 (шоу рум), БЦ "Беарс". В случае отсутствия в магазине конкретной товарной позиции, просьба заранее согласовать с менеджером оперативную доставку этого товара с нашего склада к Вашему приезду.
  • Режим работы: Понедельник - Пятница с 11.00 до 19.00 Доставка по Киеву курьером (Понедельник - Воскресенье не позднее 21.00)

Доставка обеспечивается по Киеву:

  • в течении суток при поступлении заказа до 15.00
  • в течении 2-х суток при поступлении заказа после 15.00

Доставка обеспечивается по Киеву:

Стоимость оплаты доставки:

  • на указанный нами номер карты Приватбанка

  • При стоимости покупки выше 400 грн. доставка малогабаритных товаров по Киеву бесплатно.

  • При стоимости покупки выше 400 грн. доставка малогабаритных товаров по Киеву бесплатно.

  • При заказе акционных товаров, доставка - 25 грн.

  • При стоимости покупки ниже 250 грн. доставка по Киеву - 35 грн.

  • Минимальная сумма заказа для доставки 200 грн.

Стоимость доставки товара по Киеву включет обслуживание подготовленными курьерами, которые доступно и доходчиво проконсультируют Вас в отношении эксплуатации, приобретаемого Вами прибора. Доставка по Украине осуществляется транспортными компаниями "Ночной экспресс", "Автолюкс", "Новая почта", "Интайм", "Укрпочта"

По Украине:

  • в день поступления оплаты при согласовании всех необходимых данных в отношении получателя (Ф.И.О., контактные телефоны, точный адрес и т.д.)
